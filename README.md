# AutoCare - Automobile Service Center Management System

## Overview
A full-stack Automobile Service Center Management System with real-time service tracking, customer approval workflow, role-based authentication, and AI-powered maintenance recommendations.

## Features

- User Registration & Login
- Vehicle Registration
- Service Request Management
- Real-time Service Updates
- Cost Estimate Approval
- Employee Dashboard
- Manager Dashboard
- Customer Dashboard
